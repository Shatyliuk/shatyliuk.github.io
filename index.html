About

Hello, my name is Eugene Shatyliuk and i like to code. I learned html/css/JS on my own. I read various information on html,css and js and also watched a lot of tutorial on Youtube.

I didn't just read and watch, but practiced what i learned, i think, this is the best way to learn to code.

I started with html, just basic stuff. I created basic html pages, without styling. Once I got the basics of html, I started CSS and started styling those basic html pages. Once I got the basics of CSS, I went towards more intermediate level html, like forms and other such stuff. Then learned how I can style those intermediate html elements.

After html and css I started with Javascript and Jquery basics from "JavaScript & jQuery: The Missing Manual" and from Youtube. After I learned JS and jQuery basics I started with DOM. I learned and created simple programs to manipulate html elements, like hover animation, onclick functions, etc. I am still in the process of learning JS.
